# Tic Tac Toe Game

This is a simple Tic Tac Toe game built using HTML, Tailwind CSS, and JavaScript. The game allows two players to take turns marking spaces in a 3Ã—3 grid until one player wins or the game ends in a draw.

## Features

Two-player mode <br/>
